


# 프로그램 개요
이 프로그램은 React Native 여행일지 어플리케이션에 서버를 제공하는 프로그램입니다. Tour API에서 제공하는 정보를 Mongo db저장하고 활용해서 API 서버를 구축합니다.

# 제공하는 기능
- 사용자 정보를 저장/조회
- Tour API에서 정보를 저장
- Tour 정보 조회
- 여행일지를 저장/조회
등의 기능을 제공합니다.
